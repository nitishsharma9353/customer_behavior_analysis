Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI. The goal was to clean, analyze, and visualize data to generate actionable business insights.

Dataset

Structured dataset (CSV/Database)

Includes multiple business-related variables

Used for EDA, SQL analysis, and dashboard creation

Tools Used

Python (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning & EDA

MySQL  – SQL queries & aggregations

Power BI – Interactive dashboard

Gamma – Presentation (PPT)

Jupyter Notebook – Analysis workflow

Project Steps

Loaded dataset in Python

Cleaned and preprocessed data

Performed EDA & visualizations

Ran SQL queries for insights

Built Power BI dashboard

Created report & PPT presentation

Dashboard

KPI overview

Trend analysis

Category/segment breakdown

Interactive filters
